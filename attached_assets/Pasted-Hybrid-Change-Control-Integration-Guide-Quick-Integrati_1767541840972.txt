Workflow After Integration

User clicks Edit on FMEA/Control template row
Navigates to dedicated edit page with all fields
User makes changes

Minor (notes only) → Saves directly
Standard (failure mode, cause, etc.) → QuickChangeDialog
Critical (S/O/D, controls, CSR) → Full ChangeWizard


Changes create a change package for tracking
Critical changes require impact analysis + approval routing
